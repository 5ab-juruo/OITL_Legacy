# OITL——Olympiad in Informatics Template Library，OI标准库
## 这是个什么东西？
`OITL` 是一个由私人完成的代码库，致力于让所有 OIer 们都用上清爽，快速的算法模板。

这是一个可以引用的库$^{[1]}$，也可以参与数据生成。

目前这个库不算特别完整。如果您的代码能力足够强$^{[2]}$，可以参与我们的开发$^{[3]}$
## 我应该怎么用它？
请参考目录下的 `guide.md`。
